# Todo List

A simple and intuitive Todo List web application built using HTML, CSS, and JavaScript. This project allows users to add, delete, and mark tasks as completedâ€”helping them stay organized and productive.

## Features

- **Add Tasks:** Users can add new tasks to their list.
- **Delete Tasks:** Easily remove tasks that are no longer needed.
- **Mark as Complete:** Click on tasks to mark them as done.
- **Clean UI:** Simple and user-friendly interface.

## Technologies Used

- **HTML5:** Structure of the application.
- **CSS3:** Styling and responsive design.
- **JavaScript (Vanilla):** Application logic and interactivity.
